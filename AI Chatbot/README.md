# AI Chatbot using FastAPI

## Overview
A simple AI chatbot using **FastAPI** and **Machine Learning** (Logistic Regression + TF-IDF).

## Features
- Intent-based chatbot
- ML model trained with sample intents
- FastAPI API for integration
- Interactive testing using Swagger UI
